## Relative versus absolute imports

Brad: Prefer imports at directory level, not the package level.
 
```python
# Don't like
import pylith
bc = pylith.boundary_conditions.BoundaryCondition

# Prefer
from .. import boundary_conditions
bc = boundary_conditions.BoundaryCondition

# Alternative
from pylith import boundary_conditions
bc = boundary_conditions.BoundaryCondition
```
